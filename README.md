# PRODIGY_SD_02 – Guessing Game Program (Java)

## Description
This project is a Java-based Guessing Game developed as part of the Software Development Internship at Prodigy Infotech.

The program generates a random number between 1 and 100 and challenges the user to guess it. The program provides feedback on whether the guess is too high or too low and continues until the correct number is guessed.

## Features
- Generates a random number between 1 and 100
- Accepts user guesses
- Provides feedback ("Too High" / "Too Low")
- Counts and displays the number of attempts
- Simple and interactive console-based game

## Technologies Used
- Java
- Random class
- Scanner class

## How It Works
1. The program generates a random number between 1 and 100
2. The user enters a guess
3. The program compares the guess with the generated number
4. It displays whether the guess is too high or too low
5. The game continues until the correct number is guessed
6. The total number of attempts is displayed

- Improved understanding of loops and conditional statements
- Practiced using the Random class
- Strengthened user input handling using Scanner
- Gained experience in building interactive console applications
